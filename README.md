## vSemVer

### Description
Semantic version module written for the V language.
The specification followed can be found [here](https://semver.org/)

### Usage
```v
v := SemVer{
    major: 0
    minor: 1
    patch: 27
    stage: .alpha
    build: '077e06b'
}
```
